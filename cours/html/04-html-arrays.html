Les tableaux en HTML

Les tableaux permettent de présenter des données de façon structurée, sous forme de lignes et de colonnes. Ils sont très utilisés pour organiser des informations, des horaires, des résultats, etc.

Structure de base d’un tableau

Un tableau HTML est défini avec la balise <table>. Il est composé de lignes (<tr>), d’en-têtes (<th>) et de cellules (<td>).

<table>
    <tr>
        <th>En-tête 1</th>
        <th>En-tête 2</th>
    </tr>
    <tr>
        <td>Cellule 1</td>
        <td>Cellule 2</td>
    </tr>
</table>

Les balises importantes

  • <table> : crée le tableau
  • <tr> : crée une ligne
  • <th> : crée une cellule d’en-tête (en gras, centrée par défaut)
  • <td> : crée une cellule de données
  • <thead> : regroupe l’en-tête du tableau
  • <tbody> : regroupe le corps du tableau
  • <tfoot> : regroupe le pied du tableau
  • <caption> : ajoute un titre au tableau

Fusion de cellules : colspan et rowspan

On peut fusionner des cellules horizontalement (colspan) ou verticalement (rowspan) :

<table>
    <tr>
        <th colspan="2">Fusion de 2 colonnes</th>
    </tr>
    <tr>
        <td rowspan="2">Fusion de 2 lignes</td>
        <td>Cellule 1</td>
    </tr>
    <tr>
        <td>Cellule 2</td>
    </tr>
</table>

Bonnes pratiques

  • Utiliser <thead>, <tbody> et <tfoot> pour structurer les tableaux complexes.
  • Ajouter une <caption> pour décrire le contenu du tableau.
  • Privilégier l’accessibilité : utiliser les balises sémantiques et penser à l’ordre de lecture.
  • Éviter les tableaux pour la mise en page : ils sont faits pour les données tabulaires !

Résumé

Les tableaux HTML sont puissants pour organiser des données. Grâce aux balises sémantiques et aux attributs comme colspan et rowspan, ils peuvent s’adapter à de nombreux besoins. N’oublie pas de les styliser avec du CSS pour les rendre plus lisibles et agréables !

Astuce : Pour des tableaux très complexes ou interactifs, il existe aussi des bibliothèques JavaScript spécialisées.
