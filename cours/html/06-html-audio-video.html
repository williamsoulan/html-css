Intégrer de l’audio et de la vidéo en HTML5

HTML5 permet d’intégrer facilement des fichiers audio et vidéo dans vos pages web, sans plugin externe. On peut aussi intégrer des vidéos de plateformes comme YouTube grâce à la balise <iframe>.

1. La balise <audio>

Permet de lire un fichier audio (musique, podcast, bruitage…).

<audio src="/assets/audio/mon-son.mp3" controls></audio>
  • controls : affiche les contrôles (lecture, pause, volume…)
  • autoplay : démarre automatiquement (souvent bloqué par les navigateurs)
  • loop : lecture en boucle
  • muted : commence en mode muet

On peut proposer plusieurs formats pour une meilleure compatibilité :

<audio controls>
    <source src="mon-son.mp3" type="audio/mpeg">
    <source src="mon-son.ogg" type="audio/ogg">
    Votre navigateur ne supporte pas l’audio HTML5.
</audio>

2. La balise <video>

Permet d’intégrer une vidéo locale ou distante.

<video src="/assets/videos/ma-video.mp4" controls width="320" poster="/assets/images/drMario.png"></video>
  • controls : affiche les contrôles
  • autoplay : démarre automatiquement
  • loop : lecture en boucle
  • muted : commence en mode muet
  • poster : image affichée avant la lecture
  • width, height : dimensions de la vidéo

Comme pour l’audio, on peut proposer plusieurs formats :

<video controls width="320" poster="miniature.jpg">
    <source src="ma-video.mp4" type="video/mp4">
    <source src="ma-video.webm" type="video/webm">
    Votre navigateur ne supporte pas la vidéo HTML5.
</video>

3. Formats supportés

  • Audio : MP3, OGG, WAV
  • Vidéo : MP4 (H.264), WebM, OGG

Vérifiez la compatibilité selon vos besoins : MDN - Formats supportés

4. Intégrer une vidéo YouTube ou externe avec <iframe>

Pour afficher une vidéo YouTube, copiez le code d’intégration fourni par YouTube ou utilisez la balise <iframe> :

<iframe width="400" height="225" src="https://www.youtube.com/embed/VIDEO_ID" frameborder="0" allowfullscreen></iframe>

Bonnes pratiques

  • Proposez plusieurs formats pour une compatibilité maximale.
  • Ajoutez toujours l’attribut controls pour l’accessibilité.
  • Utilisez poster pour les vidéos afin d’afficher une image avant lecture.
  • Respectez les droits d’auteur pour les médias utilisés.
