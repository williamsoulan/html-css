Gérer le texte en CSS

CSS permet de contrôler l’apparence du texte de nombreuses façons. Voici les propriétés les plus courantes:

font-size

Définit la taille du texte.

.exemple-font-size {
  font-size: 2em;
}

Texte en 2em

font-family

Définit la police utilisée.

.exemple-font-family {
  font-family: 'Courier New', Courier, monospace;
}

Texte en Courier New

font-style

Permet de mettre le texte en italique.

.exemple-font-style {
  font-style: italic;
}

Texte en italique

font-weight

Contrôle l’épaisseur du texte (gras, normal, etc.).

.exemple-font-weight {
  font-weight: bold;
}

Texte en gras

text-decoration

Ajoute une décoration (souligné, barré, etc.).

.exemple-text-decoration {
  text-decoration: underline wavy red;
}

Texte souligné ondulé rouge

text-align

Aligne le texte horizontalement.

.exemple-text-align {
  text-align: center;
}

Texte centré

float

Permet de faire flotter un élément à gauche ou à droite du flux normal.

.exemple-float {
  float: right;
  width: 200px;
  background: #e3e9f3;
  margin: 0 0 12px 12px;
  padding: 8px;
  border-radius: 6px;
}
Boîte flottante à droite

Ce texte s'écoule autour de la boîte flottante.

@font-face

Permet d’utiliser une police personnalisée.

@font-face {
  font-family: 'MaPolicePerso';
  src: url('../../assets/fonts/IndieFlower-Regular.ttf') format('truetype');
}
.exemple-font-face {
  font-family: 'MaPolicePerso', cursive;
}
