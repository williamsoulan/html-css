Gérer les bordures en CSS

Les bordures permettent de délimiter, mettre en valeur ou styliser les éléments HTML. CSS offre de nombreuses propriétés pour personnaliser l’apparence des bordures.

1. border : Propriété raccourcie

Permet de définir en une seule ligne l’épaisseur, le style et la couleur de la bordure.

div {
    border: 2px solid #2a4d7a;
}
border: 2px solid #2a4d7a;

2. border-width, border-style, border-color

Pour contrôler séparément l’épaisseur, le style et la couleur :

div {
    border-width: 4px;
    border-style: dashed;
    border-color: #e74c3c;
}
Bordure en pointillés rouges

3. Bordures individuelles

On peut cibler chaque côté : border-top, border-right, border-bottom, border-left

div {
    border-top: 4px solid #3498db;
    border-bottom: 4px dotted #e67e22;
}
Bordure haut et bas personnalisées

4. border-radius : Bords arrondis

Permet d’arrondir les coins d’un élément.

div {
    border: 2px solid #2a4d7a;
    border-radius: 16px;
}
border-radius: 16px

5. outline : Contour externe

Ajoute un contour autour de l’élément, sans décaler la mise en page.

div {
    outline: 3px dashed #b30000;
}
outline: 3px dashed #b30000

Styles de bordure disponibles

  • solid : plein
  • dashed : pointillés
  • dotted : pointillés ronds
  • double : double ligne
  • groove, ridge, inset, outset : effets 3D
  • none : pas de bordure

Bonnes pratiques

  • Utilisez les bordures pour structurer ou mettre en valeur, sans surcharger la page.
  • Combinez border-radius pour des boutons ou cartes modernes.
  • Privilégiez le contraste pour l’accessibilité.
