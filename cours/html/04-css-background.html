Gérer les backgrounds en CSS

Les backgrounds (arrière-plans) permettent d'améliorer l'apparence de vos pages web. CSS offre de nombreuses propriétés pour personnaliser les couleurs, images et comportements d'arrière-plan.

1. background-color : Couleur de fond

Permet de définir la couleur d'arrière-plan d'un élément.

body {
    background-color: #f8f9fa;
}
Exemple de background-color

2. background-image : Image de fond

Permet d'ajouter une ou plusieurs images en arrière-plan.

div {
    background-image: url("/assets/images/drMario.png");
}
Exemple de background-image

On peut aussi utiliser des dégradés CSS :

div {
    background-image: linear-gradient(90deg, #3498db, #e74c3c);
}
Exemple de background dégradé

3. background-repeat : Répétition de l'image

Contrôle si l'image de fond se répète (par défaut oui).

div {
    background-image: url("/assets/images/drMario.png");
    background-repeat: no-repeat;
}
repeat (défaut)
no-repeat

4. background-position : Position de l'image

Permet de placer l'image de fond à un endroit précis.

div {
    background-image: url("/assets/images/drMario.png");
    background-repeat: no-repeat;
    background-position: right bottom;
}
background-position: center
background-position: right bottom

5. background-size : Taille de l'image

Permet de redimensionner l'image de fond.

div {
    background-image: url("/assets/images/drMario.png");
    background-size: cover;
}
background-size: auto
background-size: cover
background-size: contain

6. background-attachment : Fixer l'arrière-plan

Définit si l'image de fond défile avec la page ou reste fixe.

div {
    background-image: url("/assets/images/drMario.png");
    background-attachment: fixed;
}
background-attachment: fixed
background-attachment: scroll

7. background : Propriété raccourcie

Permet de définir toutes les propriétés d'arrière-plan en une seule ligne.

div {
    background: url("/assets/images/drMario.png") no-repeat center/cover #f8f9fa;
}

Ordre : background-color | background-image | background-repeat | background-position / background-size | background-attachment

Bonnes pratiques

  • Optimisez la taille de vos images pour de meilleures performances.
  • Utilisez les dégradés CSS pour des effets modernes sans images.
  • Testez le rendu sur différents écrans et tailles de fenêtre.
  • Pensez à l’accessibilité : gardez un bon contraste entre le fond et le texte.
