Les sélecteurs en CSS

Les sélecteurs CSS permettent de cibler des éléments HTML afin de leur appliquer des styles. Voici les principaux types de sélecteurs :

Sélecteur de type

Il cible tous les éléments d'un certain type (balise).

p {
  color: blue;
}

Sélecteur de classe

Il cible tous les éléments ayant une classe spécifique.

.important {
  font-weight: bold;
}

Sélecteur d'identifiant

Il cible un élément unique avec un identifiant.

#unique {
  background: yellow;
}

Sélecteur descendant

Il cible les éléments à l'intérieur d'un autre élément.

article p {
  margin-left: 20px;
}

Les sélecteurs avancés

Sélecteur descendant

Le sélecteur descendant cible tous les éléments qui se trouvent à l'intérieur d'un autre élément, peu importe leur niveau d'imbrication.

section p {
  color: green;
}

Section

Paragraphe dans une section (ciblé).

Paragraphe dans un div dans une section (ciblé aussi).

Sélecteur adjacent (A+B)

Le sélecteur adjacent A + B cible l'élément B qui suit immédiatement l'élément A (ils doivent être au même niveau).

h2 + p {
  color: orange;
  font-style: italic;
}

Un titre

Ce paragraphe est immédiatement après un h2 (ciblé).

Un autre titre

Un span (non ciblé)

Ce paragraphe n'est pas immédiatement après un h2 (non ciblé).

Sélecteur d'attribut

Le sélecteur d'attribut cible les éléments possédant un attribut ou une valeur d'attribut spécifique.

a[target="_blank"] {
  color: #0077cc;
  text-decoration: underline wavy;
}
