Gérer les couleurs en CSS

CSS permet de personnaliser les couleurs du texte, des fonds et de nombreux autres éléments. Voici les principales propriétés et notations :

color

Permet de changer la couleur du texte.

.exemple-color {
  color: #e74c3c;
}

Texte en rouge (color)

background-color

Permet de changer la couleur de fond d’un élément.

.exemple-bgcolor {
  background-color: #f9e79f;
  padding: 8px;
  border-radius: 6px;
}

Fond jaune pâle (background-color)

Notation hexadécimale

On peut définir une couleur avec un code hexadécimal, commençant par # suivi de 3 ou 6 chiffres/lettres.

.exemple-hex {
  color: #3498db;
}

Texte en bleu (hexadécimal)

La fonction rgb()

Permet de définir une couleur en précisant la quantité de rouge, vert et bleu (de 0 à 255).

.exemple-rgb {
  color: rgb(46, 204, 113);
}

Texte en vert (rgb)

La fonction rgba()

Comme rgb(), mais avec un canal alpha (transparence) de 0 (transparent) à 1 (opaque).

.exemple-rgba {
  background-color: rgba(52, 152, 219, 0.3);
  padding: 8px;
  border-radius: 6px;
}

Fond bleu transparent (rgba)
